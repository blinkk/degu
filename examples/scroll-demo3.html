This is an example of using VectorDom timeline feature. You can control vector positions and rotation via a timeline.

With VectorDom you can also set css variables. Notice how the parent element has a background changing. This is set via css variables that are updated with VectorDom.

This demo can also serve to demonstrate gimble lock of eular degrees. VectorDom uses Quaternions to avoid this but you can enable eularRotationAsRotationMatarix to true to see the effects of gimble lock on the flower image.
