getViewportY

  • getViewportY(element: HTMLElement): number

  • Returns the visible distance between the top of the given element and the top of the viewport.

    Useful for specific scroll-based effects and in-view checks that cannot be easily expressed via a configuration.

    function getEffectPercent() {
   const start = getSomeHeaderHeight();
   const end = start + getViewportHeight() - getSomeFooterHeight();
   const distanceFromRoot = getViewportY(this.el);
   return mathf.inverseLerp(-1 * start, -1 * end, -1 * distanceFromRoot);
}
