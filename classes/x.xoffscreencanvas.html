Class XOffScreenCanvas

A general canvas that exists in memery for x-engine to perform offscreen calculatons. Note this is NOT the same as the buffer rendering canvas TODO (uxder) to be implemented. But more a shared, singleton to canvas that can be drawn on to do calculations.

Note that since this is shared resource, there is no guarantee of state. Objects using it should clean up.
