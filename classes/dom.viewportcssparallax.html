Class ViewportCssParallax

A class that creat a viewport css parallax. This is the base controller for viewport-css-parallax so check documentation there. `

// Create the instance and set the root element const parallaxer = new ViewportCssParallaxer();

// Add your settings. const settings = { rootElement: document.getElementById('hoho'), debug: false, lerp: 0.18, damp: 0.18, elmenentBaseline: 0 }

parallaxer.init(settings, [ { id: '--x', progress: [ {from: 0, to: 1, start: '0px', end: '0px'} ] } ] )

// Later Dispose. parallaxer.dispose(); `
