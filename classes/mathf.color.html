Static cssToRgba

  • cssToRgba(css: string): ColorRGBA | null
    • Defined in mathf/color.ts:215

    Converts a css like rgba string to an RGBA object.

    
color.cssToRgba('#FFFFFF') // { r: 255, b: 255, g: 255, a: 1}
color.cssToRgba('rgba(255, 255, 255, 0.3)') // { r: 255, b: 255, g: 255, a: 0.3}
color.cssToRgb('rgb(255, 255, 255)') // { r: 255, b: 255, g: 255, a: 1}

color.cssToRgba(20) // null
color.cssToRgba('hello') // null

Static rgbToHex

  • rgbToHex(rgb: ColorRGB | ColorRGBA): string
    • Defined in mathf/color.ts:167

    Converts an rgb out to a standard hex value.

    Modified version of: https://stackoverflow.com/questions/5623838/rgb-to-hex-and-hex-to-rgb

    
color.rgbToHex({r: 255, g: 255, b: 255}); // #FFFFFF
color.rgbToHex({r: 0, g: 0, b: 0}); // #000000

// You can pass RGBA but it will simply get ignored.
color.rgbToHex({r: 255, g: 255, b: 255, a: 1}); // #FFFFFF

    Parameters

    • rgb: ColorRGB | ColorRGBA

    Returns string

Static rgbaLerp

  • rgbaLerp(start: ColorRGBA, end: ColorRGBA, t: number): ColorRGBA
    • Defined in mathf/color.ts:124

    A stardard lerp between two rgba values. This simple algo is a straight line interpolation and isn't always going to guarantee the right hue between values. In short, it's not gonna be very colorful between values.

    
let a = { r: 0, g: 0, b: 0, a: 0};
let b = { r: 255, g: 255, b: 255, a: 1};
let lerp = color.rgbaLerp(a, b, 0.5);

    Parameters

    • start: ColorRGBA

      The starting rgba color.

    • end: ColorRGBA

      The ending rgba color.

    • t: number

      The current progress.j

    Returns ColorRGBA
