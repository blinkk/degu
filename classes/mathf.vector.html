Class Vector

A basic vector class that can be used for 2d or 3d. When using for 2d, the z defaults to 0.


let v = new Vector(0,0,0);
let v2 = new Vector(2,1,0);

v.add(v2);
let v3 = v.clone();
v3.scale(5);

v3.dot(v2);

Static fromAngle

  • fromAngle(angle: number, length?: number): Vector
    • Defined in mathf/vector.ts:222

    Creates a new 2d Vector with a specific angle and length. This is useful to create a directional vector.

    Example:

    
 let origin = new Vector(10, 20);
 // Create a ray that goes 90 degrees down in 100 length.
 let ray = Vector.fromAngle(mathf.degreeToRadian(90), 100);

 // Combine the origin and ray to get the endpoint (the tip of the ray).
 let endPoint = Vector.add(origin, ray);

    Parameters

    • angle: number

      Angle in radians

    • Default value length: number = 100000

      The length, magnitude of the vector. Defaults to a semi-infinite length.

    Returns Vector

  • fromRotationMatrixIV(m: MatrixIV): Vector
    • Defined in mathf/vector.ts:705

    Creates a Eular vector from a rotationalMatrix. This conversion is better avoided for different reasons. The pitch (y) will be limited to +-90.

    There are also cases where accuracy suffers so use with caution.

    Yaw, Roll will be +-180. Pitch will be +- 90.

  let eularVector = Vector.fromRotationMatrixIV(matrix);

Static getXyzRotationTo

  • getXyzRotationTo(originVector: Vector, targetVector: Vector): number[]
    • Defined in mathf/vector.ts:274

    Given a targetVector, this will find the rotation angles of each axis which you can then later use to on rotateX, rotateY, rotateZ values on a matrix or rotational vector.

    Basic Example:

    
let origin = Vector.ZERO;
let target = new Vector(10,15,20);
let angles = Vector.getXyzRotationTo(origin, target);

angles[0] // 1.95.. -- radian x rotation value
angles[1] // 2.16.. -- radian y rotation value
angles[2] // 2.40.. -- radian z rotation value

    See for a reference but the below is not using this: https://stackoverflow.com/questions/48532207/get-xyz-rotation-from-pvector

    Parameters

    • originVector: Vector
    • targetVector: Vector

    Returns number[]

    Array An array with three numbers, angleX, angleY and angleZ in radians.
